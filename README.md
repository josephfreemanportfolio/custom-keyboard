# custom-keyboard

## Why keyboard?
This is my first project for Hack Club. I chose to make a keyboard because I've wanted to make a custom keyboard and it was listed as a possible project.

## Layout + Parts and reasoning
- 75% keyboard
  - Gives access to all necessary keys + fn keys
- Knob
  - Better control of volume
- Mini screen
  - Screen gimmick allows for animations
- 3d printed case + wood veneer
  - 3d printing is much cheaper and a wood veneer mimmicks the aesthetic of wood

- Backglow TBD
- Switches TBD(possibly linear)
